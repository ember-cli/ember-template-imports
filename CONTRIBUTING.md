# How To Contribute

## Installation

* `git clone <repository-url>`
* `cd ember-template-imports`
* `yarn install`

## Building

* `yarn build`

## Linting

* `yarn lint:hbs`
* `yarn lint:js`
* `yarn lint:js -- --fix`

## Running tests

* `yarn test:jest` – Runs jest tests
* `yarn test:ember` – Runs ember tests
